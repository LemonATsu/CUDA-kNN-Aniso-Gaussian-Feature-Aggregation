# kNN Feature aggregation in CUDA

This is a CUDA PyTorch binding of kNN feature aggregation with anisotropic gaussian weighting.

## Installation
Preqrequisites:
```
Python >= 3.8
CUDA >= 11.0
PyTorch >= 2.0.0
```
